1. RESUMEN EJECUTIVO

Aplicación web profesional de calculadora de presupuestos de aire acondicionado en Mallorca, desarrollada como una Single Page Application con flujo de 6 pasos. La aplicación permite a los clientes finales obtener presupuestos cerrados de forma inmediata, seleccionando tipo de servicio, equipo, características del espacio y extras.

URL de producción: https://climaya.adsventas.es

4. TAREAS PENDIENTES

⚠️ Fichas Técnicas — Acción requerida por el cliente

La funcionalidad de subida de fichas técnicas está implementada y operativa. El cliente debe facilitar los archivos PDF o imágenes de las fichas técnicas de cada modelo, indicando a qué modelo corresponde cada una.

Cómo subir una ficha:

  1. Acceder a https://climaya.adsventas.es/admin
  2. Ir a la pestaña "Modelos"
  3. En la columna "Ficha Técnica", hacer clic en "📎 Subir" del modelo correspondiente
  4. Seleccionar el archivo PDF o imagen
  5. La ficha quedará vinculada al modelo y aparecerá en el presupuesto del cliente
